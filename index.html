SHASMIN

Asistente inteligente con interfaz HUD futurista, voz, visión y procesamiento multimodal.

SHASMIN es una asistente de escritorio diseñada para automatizar tareas, procesar archivos, interactuar por voz, analizar imágenes y documentos, controlar acciones del sistema y ofrecer una experiencia visual premium inspirada en interfaces holográficas.

Funciones principales

Interfaz HUD

Diseño visual futurista con orbe de partículas, branding RGanen Industries y experiencia inmersiva.

Multimodal

Puede analizar imágenes, leer PDFs, procesar documentos y trabajar con archivos cargados por el usuario.

Voz y automatización

Interacción por micrófono, comandos escritos, acciones del sistema, navegador, YouTube y tareas locales.

Licencia

Trial de 30 días

Al instalar SHASMIN por primera vez, se activa automáticamente una prueba gratuita de 30 días.

API propia

Cada usuario utiliza su propia API Key de Gemini. SHASMIN no incluye claves privadas del desarrollador.

Activación offline

Al finalizar la prueba, el usuario puede activar SHASMIN mediante un archivo de licencia personalizado.
